An agentic software workbench

Pi brings planning, code editing, shell execution, document parsing, web research, and agent skills into one careful implementation loop.

Core promise

Give the agent enough context, tools, and verification paths to make real project changes safely.

System model

Pi coordinates the whole coding loop

The agent can inspect, reason, act, and verify without leaving the project context.

Files

Read, write, edit, search, and track project state with deterministic tool calls.

Commands

Run checks, tests, scripts, local servers, and smoke workflows directly from the repo.

Skills

Load focused instructions for plans, reviews, browser work, SSH, documents, and research.

Feedback

Use Claude, subagents, previews, and validation to challenge assumptions before shipping.

Workflow

A disciplined path from prompt to verified change

Phase Agent behavior Typical tools
Understand Read docs, source, tests, plans, and current git state before acting. read, grep, find, git diff
Plan Ask clarifying questions, compare options, and get critique for risky work. create-plan, Claude feedback
Implement Make focused edits, keep the codebase clean, and avoid unnecessary abstractions. edit, write, bash
Verify Run targeted tests first, then the full project check before reporting. pnpm test, pnpm check
