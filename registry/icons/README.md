# Icons

Curated [Lucide](https://lucide.dev) icon subset (ISC license) as individual `<symbol>` files plus `manifest.json`.

Files are generated by `scripts/icons-sync.mjs` from the pinned `lucide-static` devDependency — do not edit symbols by hand; adjust the curation list in the script and rerun it.

Delivery is an inline sprite: `slidesls init` injects a hidden `<svg class="ls-sprite">` into the deck, and `slidesls icons sync --dir <deck>` rewrites it to exactly the icons the deck references via `<use href="#ls-i-<name>">`. External sprite files are not used because `<use href="file.svg#id">` is blocked over `file://`.
