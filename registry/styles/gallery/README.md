# Gallery

Swiss, white, huge type: black text on white, exactly one vivid accent (International Orange), square edges, thin rules, and whitespace doing the work texture would do elsewhere. Motion is precise horizontal slides with zero bounce.

## Activation

```html
<html lang="en" data-ls-style="gallery">
  ...
  <link rel="stylesheet" href="./slidesls/registry/fonts/space-grotesk/font.css" />
  <link rel="stylesheet" href="./slidesls/registry/fonts/inter/font.css" />
  <link rel="stylesheet" href="./slidesls/registry/fonts/jetbrains-mono/font.css" />
  <link rel="stylesheet" href="./slidesls/registry/styles/gallery/style.css" />
</html>
```

## Voice

- Fewer, bigger: one idea per slide in display type; supporting copy stays small and quiet.
- The orange accent is rationed — one element per slide at most.
- No decoration: rules, folio numerals, and scale contrast are the entire visual system.

## When not to use

- Dense data or technical walkthroughs (use `terminal`).
- Warm narrative teaching decks (use `editorial`).
