# Terminal

Dense, engineered, phosphor: near-black surfaces, a single green phosphor accent, JetBrains Mono display type over Inter body, hard 1px borders, and an instant steppy motion signature.

## Activation

```html
<html lang="en" data-ls-style="terminal">
  ...
  <link rel="stylesheet" href="./slidesls/registry/fonts/jetbrains-mono/font.css" />
  <link rel="stylesheet" href="./slidesls/registry/fonts/inter/font.css" />
  <link rel="stylesheet" href="./slidesls/registry/styles/terminal/style.css" />
</html>
```

## Voice

- Density is welcome: tables, code, and status readouts are at home here.
- One accent only — phosphor green does all emphasis work; never add warm colors.
- Motion is instant and stepped: content blinks in like a redraw, nothing drifts or bounces.

## When not to use

- Warm human storytelling, culture, or education decks (use `editorial`).
- Executive or investor audiences expecting calm authority (use `boardroom`).
