# Asymmetric Feature

Creative layout with a bold statement beside stacked supporting content.

## Usage

Use `.ls-asymmetric-feature` as the slide body or direct slide-inner child.

Key classes/attributes:

- `.ls-asymmetric-feature__statement`
- `.ls-asymmetric-feature__headline`
- `.ls-asymmetric-feature__stack`
- `.ls-asymmetric-feature__annotation`
- `data-ls-anchor="statement"` enables progressive anchor-positioning where supported

## Copy

Copy `asymmetric-feature.css` after `core/base` styles.
