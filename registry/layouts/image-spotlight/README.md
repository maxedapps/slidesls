# Image Spotlight

Large visual layout with caption, overlay, or side annotation.

## Usage

Use `.ls-image-spotlight` as the slide body or direct slide-inner child.

Key classes:

- `.ls-image-spotlight__media`
- `.ls-image-spotlight__content`
- `.ls-image-spotlight__caption`
- `.ls-image-spotlight__label`

Images and videos inside the media region are fitted with `object-fit: cover`.

## Copy

Copy `image-spotlight.css` after `core/base` styles.
