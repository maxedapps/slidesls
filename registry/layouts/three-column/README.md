# Three Column

Equal/weighted feature, pillar, option, or lesson columns.

## Usage

Root `.ls-three-column`; children `.ls-three-column__item`, `__header`, `__title`, `__body`, `__footer`. Attributes: `data-ls-align`, `data-ls-valign`, `data-ls-density`, `data-ls-dividers`. Variables: `--ls-three-column-gap`, `--ls-three-column-min-inline`, `--ls-three-column-max-inline`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
