# Code Explainer

Technical teaching layout with code and explanatory notes.

## Usage

Root `.ls-code-explainer`; regions `__code`, `__notes`, `__steps`, `__summary`. Attributes: `data-ls-ratio`, `data-ls-density`, `data-ls-valign`. Variables: `--ls-code-explainer-gap`, `--ls-code-explainer-code-size`, `--ls-code-explainer-notes-size`.
