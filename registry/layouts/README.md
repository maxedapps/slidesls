# Layouts

Reusable slide layout patterns.

Current items:

- `title-hero/` — opening slide with hero title and visual area.
- `detail-split/` — asymmetric explanation slide with visual + stack.
- `centered-statement/` — sparse high-impact centered message slide.
- `section-divider/` — chapter or section transition slide.
- `two-column/` — reliable two-region professional layout.
- `comparison-grid/` — option, before/after, or pros/cons grid.
- `asymmetric-feature/` — bold statement beside stacked supporting content.
- `image-spotlight/` — large visual with caption, overlay, or side annotation.

Layouts depend on `core/base` and may pair with components, but reveal animation is an optional consumer choice.
