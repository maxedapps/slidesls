# Metric Dashboard

Dashboard composition for a headline insight plus metric/card regions.

## Usage

Root `.ls-metric-dashboard`; regions `__hero`, `__metrics`, `__panel`, `__footer`. Attributes: `data-ls-variant`, `data-ls-density`, `data-ls-valign`. Variables: `--ls-metric-dashboard-gap`, `--ls-metric-dashboard-hero-size`, `--ls-metric-dashboard-metric-min-inline`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
