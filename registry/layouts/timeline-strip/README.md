# Timeline Strip

Full-slide horizontal roadmap or process strip.

## Usage

Root `.ls-timeline-strip`; regions `__intro`, `__track`, `__item`, `__marker`, `__title`, `__text`. Attributes: `data-ls-density`, `data-ls-progress`, `data-ls-align`. Variables: `--ls-timeline-strip-columns`, `--ls-timeline-strip-gap`, `--ls-timeline-strip-accent`.

## Copy
