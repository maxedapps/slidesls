# Quote Feature

Full-slide quote/testimonial/editorial layout.

## Usage

Root `.ls-quote-feature`; regions `__quote`, `__mark`, `__text`, `__source`, `__context`, `__media`. Attributes: `data-ls-align`, `data-ls-valign`, `data-ls-variant`. Variables: `--ls-quote-feature-max-inline`, `--ls-quote-feature-gap`, `--ls-quote-feature-mark-size`. Use semantic `blockquote`, `footer`, and `cite`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
