# Two Column

Reliable two-region professional layout.

## Usage

Use `.ls-two-column` around two regions.

Key classes:

- `.ls-two-column__main`
- `.ls-two-column__aside`
- `.ls-two-column__stack`

Attributes:

- `data-ls-ratio="wide-main|wide-aside"`
- `data-ls-valign="start|center|space-between|stretch"`

Variables:

- `--ls-two-column-main`
- `--ls-two-column-aside`
- `--ls-two-column-gap`
- `--ls-two-column-align`
- `--ls-two-column-align-content`

## Copy

Copy `two-column.css` after `core/base` styles.
