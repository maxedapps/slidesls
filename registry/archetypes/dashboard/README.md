# Dashboard

An operational read-out: three to five tiles in the filling dashboard grid, mixing boxed surfaces (for metrics that need a sentence), bare stats (for numbers that speak alone), and a chart when a shape is honest. The slide title makes the call; the tiles are the evidence.

## Contract

- tiles: 3–5, drawn from `ls-surface`, `ls-stat`, and `ls-chart` — kept flat, never nested
- tileTitle: ≤ 8 words on surface tiles

## Motion

Tiles enter with the core stagger; a dashboard never needs `data-step`.

## When not to use

- One number carries the story: use `archetypes/big-stat` and give it the whole slide.
- Every tile would be an identical boxed panel: vary boxed and unboxed tiles, or switch to `components/table` so like values scan as rows.
