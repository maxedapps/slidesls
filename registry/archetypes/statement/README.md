# Statement

One claim at display size, with at most one quiet support line. This is the pause button of a deck — the slide where nothing competes with the sentence. The surplus whitespace is the design, not a gap to fill.

## Contract

- claim: exactly one, ≤ 14 words, written as a verdict
- support: optional, ≤ 16 words

## Motion

The core entrance stagger carries the claim and support; a statement never needs `data-step`.

## When not to use

- The claim needs badges or a figure: that is an opener — use `archetypes/title-hero`.
- The sentence is really a number: use `archetypes/big-stat` and let the value be the type.
