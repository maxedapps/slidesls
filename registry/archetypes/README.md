# Archetypes

Complete slide patterns: snippet(s) + content contract + variants, with motion already wired and zero placeholder visuals.

Every archetype ships a `snippet.html` marked `data-ls-archetype="<name>"`, a content contract (slot counts and word ranges in `registry-item.json`), `intent[]` metadata for discovery, and — when it has a figure slot — a no-figure variant.

Items in this group carry `status: "preview"` until the v2 vocabulary ships (0.8.0).
