# Fonts

Shared vendored webfont pool for slidesls v2 styles. Each family directory holds latin variable woff2 files plus that family's own `OFL.txt` (families have separate copyright holders, so each ships its own license text).

Directories are generated by `scripts/fonts-sync.mjs` from pinned `@fontsource-variable/*` devDependencies — do not edit files here by hand; rerun the sync script instead.

Styles reference families via `registryDependencies`, so a family shared by several styles is copied into a deck exactly once.
