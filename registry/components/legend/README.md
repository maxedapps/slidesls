# Legend

Accessible keyed legend for diagrams and visual explanations.

## Usage

Root `.ls-legend`, usually `ul`, `ol`, or `dl`; children `__item`, `__marker`, `__label`, `__description`. Attributes: `data-ls-orientation`, `data-ls-density`, `data-ls-variant`. Variables: `--ls-legend-gap`, `--ls-legend-marker-size`, `--ls-legend-marker-color`. Include text labels and shape/pattern cues; do not rely on color alone.

## Copy
