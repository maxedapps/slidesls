# Logo Strip

Logo, partner, tool, or ecosystem rows/grids.

## Usage

Root `.ls-logo-strip`; children `.ls-logo-strip__item`, `__mark`, `__label`. Attributes: `data-ls-density`, `data-ls-align`, `data-ls-variant`. Variables: `--ls-logo-strip-gap`, `--ls-logo-strip-item-min-inline`, `--ls-logo-strip-logo-block`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
