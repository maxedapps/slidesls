# components/card

Flat feature and explanation cards for slide content.

Requires `core/base`.

Useful classes:

- `.ls-card`
- `.ls-card--row`
- `.ls-card__body`
- `.ls-card__title`
- `.ls-card__text`

Sizing variables: `--ls-card-padding`, `--ls-card-title-size`, `--ls-card-text-size`. Compact slide density scopes smaller defaults for dense card grids.
