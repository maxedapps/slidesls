# Bullet List

Styled bullets, checklists, and process lists.

## Usage

Use semantic `ul` or `ol` markup where possible.

Key classes/attributes:

- `.ls-bullet-list`
- `.ls-bullet-list__item` for non-list markup
- `data-ls-variant="check"`
- `data-ls-density="dense"`
- `--ls-bullet-gap`

## Copy

Copy `bullet-list.css` after `core/base` styles.
