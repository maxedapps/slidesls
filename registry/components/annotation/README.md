# Annotation

Label, note, or callout block for diagrams and screenshots.

## Usage

Root `.ls-annotation`; children `__marker`, `__body`, `__title`, `__text`, `__meta`. Attributes: `data-ls-tone`, `data-ls-variant`, optional `data-ls-placement="floating"`. Variables: `--ls-annotation-accent`, `--ls-annotation-gap`, `--ls-annotation-marker-size`. Defaults to normal flow.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
