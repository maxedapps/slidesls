# Highlight Text

Inline emphasis for important phrases, values, and code-adjacent text.

## Usage

Use `.ls-highlight-text` on `<mark>` or inline elements. Attributes: `data-ls-variant`, `data-ls-tone`. Variables: `--ls-highlight-text-accent`, `--ls-highlight-text-opacity`, `--ls-highlight-text-thickness`. Static only; animation belongs to `animations/highlight`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
