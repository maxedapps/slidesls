# Divider

Internal content separator for dense slides.

## Usage

Use `.ls-divider`, preferably on `<hr>` for semantic separation. Optional `.ls-divider__label`. Attributes: `data-ls-orientation`, `data-ls-variant`, `data-ls-density`. Variables: `--ls-divider-color`, `--ls-divider-thickness`, `--ls-divider-gap`.
