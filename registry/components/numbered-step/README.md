# Numbered Step

Step item/card for tutorials, processes, and agendas.

## Usage

Use `.ls-numbered-step` for each item; optional `.ls-numbered-steps` group. Classes: `__number`, `__title`, `__text`, `__meta`. Attributes: `data-ls-variant`, `data-ls-status`. Variables: `--ls-numbered-step-accent`, `--ls-numbered-step-number-size`, `--ls-numbered-step-gap`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
