# Image Card

Image/media case-study card with caption and metadata.

## Usage

Root `.ls-image-card`, preferably `figure` or `article`; children `__media`, `__content`, `__eyebrow`, `__title`, `__text`, `__meta`, `__badge`. Attributes: `data-ls-variant`, `data-ls-ratio`, `data-ls-density`. Variables: `--ls-image-card-aspect`, `--ls-image-card-gap`, `--ls-image-card-media-block`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
