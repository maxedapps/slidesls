# Callout

Highlighted note, warning, insight, or takeaway.

## Usage

Key classes/attributes:

- `.ls-callout`
- `.ls-callout--with-icon` (only when an icon column is present)
- `.ls-callout__icon`
- `.ls-callout__body`
- `.ls-callout__title`
- `.ls-callout__text`
- `data-ls-tone="warning" | "success" | "danger"`
- `--ls-callout-accent`
- `--ls-callout-padding`
- `--ls-callout-title-size`
- `--ls-callout-text-size`

## Copy

Copy `callout.css` after `core/base` styles.
