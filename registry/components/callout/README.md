# Callout

Highlighted note, warning, insight, or takeaway.

## Usage

Key classes/attributes:

- `.ls-callout`
- `.ls-callout__icon`
- `.ls-callout__body`
- `.ls-callout__title`
- `.ls-callout__text`
- `data-ls-tone="warning" | "success"`
- `--ls-callout-accent`

## Copy

Copy `callout.css` after `core/base` styles.
