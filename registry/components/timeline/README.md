# Timeline

Reusable sequence component for processes and milestones.

## Usage

Use `.ls-timeline` on an ordered list. Each `.ls-timeline__item` must contain `.ls-timeline__marker` and `.ls-timeline__body`; put `.ls-timeline__title`, optional `.ls-timeline__meta`, and `.ls-timeline__text` inside the body. Attributes: `data-ls-orientation`, `data-ls-density`, `data-ls-progress`. Variables: `--ls-timeline-gap`, `--ls-timeline-marker-size`, `--ls-timeline-accent`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
