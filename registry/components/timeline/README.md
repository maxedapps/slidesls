# Timeline

Reusable sequence component for processes and milestones.

## Usage

Use `.ls-timeline` on an ordered list. Classes: `.ls-timeline__item`, `__marker`, `__title`, `__meta`, `__text`. Attributes: `data-ls-orientation`, `data-ls-density`, `data-ls-progress`. Variables: `--ls-timeline-gap`, `--ls-timeline-marker-size`, `--ls-timeline-accent`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
