# Templates

Paste-ready slide skeleton snippets composed from utilities and standalone components.

Current items:

- `title-hero/`
- `section-divider/`
- `split/`
- `split-diagram/`
- `three-cards/`
- `code-plus-notes/`
- `metric-dashboard/`

Templates expose HTML through `snippets` metadata and `slidesls inspect <template> --json`. They intentionally do not copy snippet HTML into deck assets through `slidesls add`.
