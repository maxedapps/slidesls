# templates/metric-dashboard

Dashboard template for metrics and progress indicators.

Use this template as a complete slide skeleton. It is composed from utilities and standalone components; it does not require a template-specific CSS class.

Keep metric labels short. The grid sizes its row to content and centers it vertically, so a single band of metrics reads as a deliberate composition; add a second row of metrics or supporting panels when the slide should carry more data.

## When not to use

- The numbers need narrative explanation — use `templates/split` with a metric in the panel column and cards for the story.
- Only one metric matters — use a hero/section slide with `data-ls-density="spacious"` instead of a one-cell dashboard.
