# templates/feature-rows

Stacked full-width rows for 3-5 one-liner points: icon and keyword left, one sentence right. The single-column grid sizes rows to content and centers the block vertically, so a short list composes as a balanced band — the alternative to `templates/three-cards` when copy is one sentence per point.

Composed from `utilities/layout` (`.ls-grid`, single column) and `components/icon-item`.

## When not to use

- Do not use feature-rows when points carry 2-4 sentences or visuals; use `templates/three-cards`.
- Do not exceed 5 rows; use `templates/icon-grid` for 4-6 very short items or split the slide.
