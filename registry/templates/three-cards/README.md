# templates/three-cards

Three-card slide template for comparing related points.

Use this template as a complete slide skeleton. It is composed from utilities and standalone components; it does not require a template-specific CSS class.

Give each card 2-4 sentences or a visual so the three columns carry real weight. The grid sizes rows to content and centers the band vertically, so cards never stretch to fill the slide.

## When not to use

- Each card would hold only a title plus one short sentence and no visual — use `templates/feature-rows` (one-liner rows) or `templates/icon-grid` (compact tiles).
- More than 4 items — they wrap into uneven rows; use `templates/icon-grid` or split the slide.
