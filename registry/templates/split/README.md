# templates/split

Two-column slide template for a visual plus supporting points.

Use this template as a complete slide skeleton. It is composed from utilities and standalone components; it does not require a template-specific CSS class.

The left panel is meant for a real visual: a diagram, screenshot, code sample, or key statement. Swap `.ls-panel--center` for `.ls-panel--frame` when the visual should keep substantial height.

## When not to use

- There is no real visual or anchor content for the panel column — use `templates/feature-rows` or `templates/three-cards` instead of a placeholder panel.
