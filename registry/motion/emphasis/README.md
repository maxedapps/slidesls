# Emphasis

Scale-and-settle for the element a step just revealed: it lands at 92% scale, peaks briefly at 103%, and settles — the number arrives with weight. Add `ls-emphasize` and `data-step` to one element; the core runtime's reveal state triggers the animation.

All kill switches apply (export, print, `prefers-reduced-motion`, `data-ls-motion="none"`).

## When not to use

- On several elements per slide — emphasis everywhere is emphasis nowhere.
- On content that already staggers in; the entrance is emphasis enough.
