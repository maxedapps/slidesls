# Motion

Optional motion recipes beyond the core defaults.

Slide transitions, auto-stagger entrances, and step reveals are core behavior (`registry/core/base/motion.css` + the runtime) — they need no item from this group. This group holds only extra recipes: emphasis, line highlight, draw-in, typing.

Items in this group carry `status: "preview"` until the v2 vocabulary ships (0.7.0).
