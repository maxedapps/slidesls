# Presets

Optional token remaps and presentation-style presets for registry consumers.

Presets are copyable registry items. They should not fork components or require this repository as a runtime dependency.

Available preset groups:

- `fonts/` — scoped font-family role remaps activated with `data-ls-font`.
- `themes/` — deck-wide visual token remaps activated with `data-ls-theme` on `<html>`.

Themes and fonts are independent axes: choose one theme for deck visual language, then optionally choose a font preset for type personality.
