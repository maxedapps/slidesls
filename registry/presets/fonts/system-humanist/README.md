# system-humanist

Scoped font preset for clean professional decks that lean on humanist system fonts.

This preset remaps body, heading, display, and label roles to a humanist sans stack without loading external font files.

## Usage

```html
<link rel="stylesheet" href="./registry/presets/fonts/system-humanist/font.css" />
<body class="ls-page" data-ls-font="system-humanist"></body>
```

For one slide only:

```html
<section class="ls-slide" data-ls-font="system-humanist"></section>
```
