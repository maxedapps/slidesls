# technical-mono

Scoped font preset for technical decks.

This preset makes labels, eyebrows, and badge-like text use the monospace stack while keeping body text readable. The code role already defaults to the monospace stack in `core/base`.

## Usage

```html
<link rel="stylesheet" href="./registry/presets/fonts/technical-mono/font.css" />
<body class="ls-page" data-ls-font="technical-mono"></body>
```

For one slide only:

```html
<section class="ls-slide" data-ls-font="technical-mono"></section>
```
