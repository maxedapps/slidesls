# Spotlight

Focus treatment for visual regions without hiding essential content.

## Usage

Use `.ls-spotlight`, `.ls-spotlight__target`, or `.ls-reveal-spotlight`. Attribute: `data-ls-variant="ring|wash|dim"`. Variables: `--ls-spotlight-accent`, `--ls-spotlight-opacity`, `--ls-spotlight-spread`. Dim mode restores full opacity in export.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
