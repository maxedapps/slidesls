# Pulse

Subtle attention pulse for markers and annotations.

## Usage

Use `.ls-pulse` for a one-shot static pulse, `.ls-pulse data-ls-loop="true"` for explicit looping, or `.ls-reveal-pulse` with reveal state. Variables: `--ls-pulse-accent`, `--ls-pulse-size`, `--ls-pulse-duration`. Reduced-motion/export disables animation.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
