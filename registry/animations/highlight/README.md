# Highlight Animation

Subtle emphasis pulse for marked text or data.

## Usage

Use `.ls-highlight` for static emphasis or `.ls-reveal-highlight` with reveal steps. Variables: `--ls-highlight-animation-duration`, `--ls-highlight-animation-accent`, `--ls-highlight-animation-spread`. Compose with `components/highlight-text` when useful.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
