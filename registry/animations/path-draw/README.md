# Path Draw

Generic SVG path drawing animation.

## Usage

Apply `.ls-path-draw` to SVG paths with `pathLength="1"`; can live inside `.ls-reveal`. Variables: `--ls-path-draw-duration`, `--ls-path-draw-ease`, `--ls-path-draw-delay`. Custom copied paths should set `pathLength="1"` or adapt the dash model.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
