# Scale In

Reveal-compatible scale/opacity entrance variant.

## Usage

Combine `.ls-reveal ls-reveal-scale-in` with `data-step`. Variables: `--ls-scale-in-start`, `--ls-scale-in-duration`, `--ls-scale-in-ease`. Load after `animations/reveal`.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation variants should load after `registry/animations/reveal`.
