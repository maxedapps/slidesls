# Connector Grow

Reveal-compatible connector line growth.

## Usage

Apply `.ls-connector-grow` to SVG connector paths with `pathLength="1"`; can live inside `.ls-reveal`. Variables: `--ls-connector-grow-duration`, `--ls-connector-grow-ease`. Uses stroke dash offset; no JS path measurement.

## Copy

Copy this item CSS after `registry/core/base` styles. Animation recipes that compose with reveals should load after `registry/animations/reveal`.
